Template Driven Form

Template-driven forms are based only on the template-directive.
template-driven forms use two-way data binding to update the data
Template driven forms are forms where we write logic, validations, controls etc, in the template part of the code (html code).
We hav to import formsModule in app.module in order for it work.
we use ngForm, ngSubmit in form tag. We use name="" and ngModel in input tags.
