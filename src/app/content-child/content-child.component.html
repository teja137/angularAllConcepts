Content Projection (child)

use ng - content to control other component i.e, parent component here
  • @ViewChild() is to select the same component single unique id
  • @ViewChildren() is to select the same component multiple same unique id's
  • @ContentChild() is to select the other component single unique id
  • @ContentChildren() is to select the same component multiple unique id's
