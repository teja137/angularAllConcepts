Data Bindings

Data binding is the communication between class and template in components
We have 4 types of Bindings
  • Interpolation
  • Property Binding
  • Event Binding
  • Two Wat Data Binding

Interpolation

Interpolation is the communication between class to template, we also can mathematical operations like add, sub, etc in template easily
It is annotated with two flower braces
