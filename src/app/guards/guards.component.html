Guards

Guards are a way to perform a check weather a user can navigate to a certain route or not

we 5 differet type of guards, used based on the requirment

  • canActivate - checks to see if a user can vist a route
  • canActivateChild - checks to see if a user can vist a routes children
  • canDeactivate - checks to see if a user can exit a route
  • canLoad (also known as lazy loading) - checks to see if a user can route to a module that is lazy loaded
  • reslove - perfomes route data reterival before route activation
